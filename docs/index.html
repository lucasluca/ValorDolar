Privacy Policy

Last updated: January 2026

This Privacy Policy applies to the ValorDolar macOS application.

Overview

ValorDolar is a macOS menu bar application designed exclusively to display the current U.S. Dollar exchange rate. The app runs locally on the user's device and does not require user accounts, registration, or authentication.

Information Collection and Use

ValorDolar does not collect, store, process, or share any personal or non-personal user data.

The application does not request access to personal information, device identifiers, location data, or system permissions beyond what is strictly necessary to display information in the macOS menu bar.

Network Usage

The app may access publicly available exchange rate information to display the current dollar value. No user-identifiable data is transmitted, logged, or stored during this process.

Third-Party Services

ValorDolar does not use third-party analytics, advertising SDKs, tracking technologies, or data collection services.

Data Retention

Since no user data is collected, no data is stored or retained.

Children's Privacy

ValorDolar does not knowingly collect any data from users of any age, including children under the age of 13.

Changes to This Privacy Policy

This Privacy Policy may be updated from time to time. Any changes will be posted on this page.
